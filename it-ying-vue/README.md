# vue学习
## 开发环境
> vue-cli (v2)
1. vue脚手架工具
- npm install --global vue-cli
2. 创建项目
- vue init webpack project-name
- vue init webpack-simple project-name

> @vue-cli (v3)
1. vue脚手架工具
- npm install -g @vue/cli
- npm install -g @vue/cli-init
2. 创建项目
- vue create project-name
- vue init webpack project-name
- vue init webpack-simple project-name

> 启动项目
1. vue init 创建的项目
- npm run dev
2. vue create 创建的项目
- npm run serve

> 打包
- npm run build

> 使用vue ui 图形化界面创建项目
- vue ui 打开创建项目的图形化界面
